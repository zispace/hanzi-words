# 汉语常用词表

数据原始来源见 `txt` 文件开头部分，词表数据有修正、合并、去重等处理。

## 格式说明

文件格式如下，文件开头以# 开头的几行是注释

```tsv
# 资源名
# 原始资料等注释说明
#
# 如果是多列，这个是TSV格式列名，第一列应是词语

词语
词语	列2	列3
```
